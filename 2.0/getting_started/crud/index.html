CRUD Operations

This page is a brief overview of performing basic create, read, update, and delete (CRUD) operations with the driver.

For more information about the .NET API, please refere to the online API Documentation.

The rest of this document assumes the following class and collection:

public class Person
{
    public ObjectId Id { get; set; }

    public string Name { get; set; }

    public int Age { get; set; }

    public string Profession { get; set; }
}

var collection = db.GetCollection<Entity>("entities");

Inserting a document

var jane = new Person { Name = "Jane McJane", Age = 24, Profession = "Hacker" };

await collection.InsertOneAsync(entity);

After insertion, the person’s Id property will contain the automatically generated identifier. For more on id generation, see the reference guide.

Finding a Document

To find all the people who are younger than 42, the following variations are valid:

var people = await collection.Find(x => x.Age < 42).ToListAsync();

// or

var filter = Builders<Person>.Filter.Lt(x => x.Age, 42);
var people = await collection.Find(filter).ToListAsync();

The lambda expressions get translated into the bson filter { Age: { $lt: 42 } }. Not everything is support via expression trees, but most (if not all) the common operations are. See the reference guide for more information.

Updating

To update a document, there are 2 methods for doing so. First, you can update specific fields. For instance, I’d like to change Tom’s profession to “Musician”.

var update = Builders<Person>.Update.Set(x => x.Profession, "Musician");
var result = await collection.UpdateOneAsync(x => x.Name == "Tom", update);

This will generate a filter of { Name: "Tom" } and an update statement of { $set: { Profession: "Musician" } }. Since we didn’t use an identifier in the query, then if there is more than one person named “Tom”, one of them will get updated.

If we simply want to replace a person’s document completely, we can use the ReplaceOneAsync method. Assuming Tom’s _id value is “550c4aa98e59471bddf68eef”:

var tom = await collection.Find(x => x.Id == ObjectId.Parse("550c4aa98e59471bddf68eef")).SingleAsync();
tom.Name = "Thomas";
tom.Age = 43;
tom.Profession = "Hacker";
var result = await collection.ReplaceOneAsync(x => x.Id == tom.Id, tom);

Identifiers in MongoDB are immutable, so you can’t replace a document with another one where the identifier is different.

Deleting

Finally, to delete Tom, we can do the following:

var result = await collection.DeleteOne(x => x.Id == tom.Id);
